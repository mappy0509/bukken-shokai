# 賃貸物件のご案内（福岡市）

スマホ最適化された4物件のスライド資料。GitHub Pages 公開用。

## 公開URL
GitHub Pages にて公開（リポジトリ Settings → Pages から確認）

## 構成
- `index.html` … 物件4件＋比較ビュー
- `style.css` … スマホ最適化スタイル
- `script.js` … タブ切り替え／スワイプ／キーボード操作

## ローカルプレビュー
```bash
cd bukken-shokai
python3 -m http.server 8080
# → http://localhost:8080
```
